    5月10、11日11:30至13:00
第九届上海大学生电视节巡回展暨第十六届东华大学学生调研中心公众开放日
在东华大学（松江校区）大学生活动中心一楼隆重举行
当秉承“精彩视觉、传播激情”的大学生电视节邂逅“协作创新，求实溯源”的调研中心，他俩又擦出怎样不一般的火花呢？



01
class
借力团队品牌“微调研”，获悉学生真期待
东华大学学生调研中心是东华大学最具特色的部门之一，其品牌项目――“微调研”更是荣登“人民网”、“中青在线”等各大平台。
这次，为了向同学们展现一场“喜大普奔”的活动盛宴，在本次大学生电视节巡回展拉开帷幕之前，东华大学学生调研中心首先以微调研的形式，在全校为本次电视节巡回展的活动形式及预期效果展开了一番深入调查。
调查结果显示，高达90%的同学对本次活动很是期待，认为能增长见识，丰富课外活动，更是一次深入了解电视节系列活动与学生调研中心学生组织的好机会。这无疑为本次巡回展活动的成功举行奠定了公众基础。

02
class
缤纷“电视直播间”，寓教于乐宽视野
近年来，“直播”一词频频出现在大众视野内，而本届电视节巡展活动，正是以大学生们喜闻乐见的“电视直播间”的活动形式。
结合团委学生调研中心特色活动公众开放日本身的活动特点，将东华大学松江校区的10个学院展台设为不同的频道，以频道栏目的形式将近几月来的社会热点和时事动态呈现给全校师生。
话题涉及体育、科技、民生、校园、医疗、娱乐等多方面，如展播“国足出线”的体育频道，解读“AlphaGo人工智能”的科技频道，普及“反家暴法”的法律频道等等。
除此之外，各展台还结合调研特色，开展丰富多彩的趣味活动，宣传调研知识，成功将“调研元素”有机嵌入电视节活动中。
在活动现场，大学生出于对频道话题的好奇，及对电视节的期待，主动参与到各类子活动中，一方面汲取到充实的调研知识，加深了对学生调研中心的了解，另一方面体验到大学生电视节巡展的生动趣味性，并通过现场观赏电视节历届优秀作品，感受佳作的非凡魅力，可谓是学习娱乐两不误。


03
class
“调”转遥控器，主题新颖贴切
获感于“上海大学生电视节”，本次公众开放日主题――“调”转遥控器，紧紧围绕“电视”这一关键词。
参与者可根据自身意愿，随意徜徉于各大频道之间，好似自己手中有把遥控器一般；而“调”字，一方面传达出本届活动公众参与的自由度，能各据所好，自由转换“频道”，另一方面，则暗含本次电视节巡展活动由“东华大学学生调研中心”这一团队承办。
本次活动以大学生电视节为契机，开展院际主题活动的形式不仅打破了传统的调研成果展示形式，丰富展示内容，更进一步提高公众开放日的影响力、趣味性和参与度，同时加强了校园文化建设。

04
class
各大学院激烈角逐，“竞合之间”共塑盛况
正所谓：“有压力才有动力。”本次活动举办过程中，新增评分环节。由团委各大老师以及活动团队成员担任评。
以10分制的计分标准，从主题明确性、活动趣味性、展台布置创新性、工作人员的积极性以及公众的参与度等多维度为各学院点评打分，以此来激发工作人员的创新思维，提升他们活动的积极性，从而带动现场活动氛围。
当然，不可否认的是，正是院际之间的“较量”共同塑造了每次大学生电视节的圆满成功，这又何尝不是一种另类“合作”呢？
