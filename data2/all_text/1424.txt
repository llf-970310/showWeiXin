今年五四青年节前夕，我校按照“组织随着青年走，团在青年群体建”的团建原则，探索推进了创新型基层团支部建团模式――东华大学大学生“小平科技创新团队”（麻类纤维生物脱胶技术产业化集成研发团队）团支部、东华大学先进电化学能源课题组团支部、东华大学易班学生团队团支部、东华大学学生合唱团团支部先后成立。

接下来让东小华带大家认识他们~  o(*￣幔*)o



01
class
东华大学大学生“小平科技创新团队“团支部
东华大学麻类纤维生物脱胶产业化集成研发团队是以纺织学院和化工生物学院博士生、硕士生和本科生为主体组成的复合型技术研发团队，团队成员由本科三、四年级，硕士，博士研究生组成。团队研发技术的作品获第十二届“挑战杯”全国大学生课外学术科技作品竞赛一等奖、第十二届上海市大学生课外学术科技作品竞赛特等奖、第十三届挑战杯全国大学生课外学术科技作品竞赛“累进创新金奖”等。该团队的技术创业项目获第八届“挑战杯”全国大学生创业计划大赛金奖、第七届上海市大学生创业计划大赛金奖。团队拥有自主知识产权授权国家发明专利15项，发表论文20多篇，获省部级以上奖励10余项。







02
class
“先进电化学能源课题组”团支部
东华大学先进电化学能源课题组围绕绿色能源开展科学研究工作，组织培养环境学院本科生、研究生参与各项科技创新活动，积极营造学术科创氛围。课题组从实验室建设及课题研究两个方面入手，研究领域涉及燃料电池及其组件研究与开发、锌空电池研究、二氧化碳电催化还原以及超级电容器等五大方面，已发表SCI论文50多篇，多次荣获全国科创类竞赛奖项。为更有效地开展青年学生联系服务工作，课题组成立“先进电化学能源课题组”团支部，目前拥有正式成员23人，其中正式党员8人，预备党员1人，共青团员14人。







03
class
东华大学易班学生团队团支部
目前，东华大学易班学生管理团队拥有正式成员45人，有3人正在党校学习。正是在这样一个平台上，管理团队的一届又一届学生在线上开展服务联系青年工作，取得了良好的成效。在学生处易班管理中心的支持下，在易班学生团队建立团支部，实现网络空间建团新尝试――统筹规划团支部线上线下团建相结合，依托团支部夯实组织基础，拓展团支部工作阵地，推进团学工作进易班。







04
class
东华大学学生合唱团团支部
东华大学学生合唱团目前拥有正式成员68人，其中党员1人，预备党员2人，共青团员65人，15人参加过党校学习。在这样一个优秀的团队中，团队成员在一起相处的时间远远超过了原来意义上的班级同学，而共同的兴趣爱好又让成员们在线上以及闲暇形成了共同话题，更易于形成组织的向心力和凝聚力。东华大学学生合唱团团支部成立以后，一方面将有利于加强团队凝聚力，同时以弘扬校园合唱文化为着力点，也将助力校园文化建设。





我校根据青年学生聚集特点，已在国旗护卫队、智力助残志愿者服务队、尚实理论学习会社团、绿手印社团等组织成立了团支部，此次成立了四个创新型基层团支部，是我校提升团支部活动的又一重要举措。提升团支部的活力不仅仅是让基层“活起来”，更是要追根溯源找到基层团支部的吸引力和凝聚力之源。
